## 一、计算机的基本硬件组成

### 1.1、CPU

+ 全称 central processing unit。核心是从程序或应用程序获取指令并执行计算。
+ CPU从系统主存中*提取*指令，然后*解码*指令的实际内容，再由CPU的相关部分*执行*该指令。
+ 个人电脑用的是Intel的CPU，苹果手机用的是ARM的CPU。两种CPU各自支持的语言不同，就是两组不同的计算机指令集instruction set。
+ CPU主要由两个部分构成：*控制单元*，*算术逻辑单元ALU*。
+ 从功能来讲，就是一系列寄存器的集合体。
+

### 1.2、内存

+ Memory 是程序和CPU沟通的桥梁，计算机中所有程序的运行都是在内存中进行的，因此内存对计算机的影响非常大，又被称为**主存**
+ 主存的作用是存放CPU的运算数据，以及与硬盘等外部存储设备交换的数据。
+ 内存内部是由各种IC电路组成的，主要分为三种存储器：随机存储器RAM、只读存储器ROM、高速缓存Cache。
+

### 1.3、主板

+ 主板上集成了各种电子元件、插槽和接口等，为CPU、内存和各种功能卡等提供了安装插槽，为各种多媒体和通信设备提供了接口。
+

### 1.4、I/O设备

+ 显示器
+ 鼠标+键盘
+

### 1.5、显卡

+ graphics card 显卡特殊是因为 显卡里有除了CPU之外的另一个 处理器，就是 GPU(graphics processing unit，图形处理器)，GPU可以做各种 计算 的工作。
+

## 二、冯诺依曼架构

### 2.1、冯诺伊曼 结构

+ 冯 提出计算机制造的三个基本原则(采用二进制、程序存储、顺序执行)和五个组成部分(输入、输出、存储、运算器、控制器)。这是冯诺依曼体系结构。
+
冯结构的计算机必须具备功能：①把需要的程序和数据送至计算机中；②具有长期记忆程序、数据、中间结果及最终运算结果的能力；③完成各种算术、逻辑运算和数据传送等数据加工处理的能力；④控制程序的走向，并能根据指令控制机器的各部件协调操作；⑤按照要求将结果输出。
+

### 2.2、哈佛架构

哈佛架构和冯架构的主要区别在于是否区分指令和数据。

### 4.2、HTTP

+ hypertext transfer protocol 超文本传输协议，拆成三个部分：超文本、传输、协议
+ 双向协议
+ http请求响应过程：假设访问某url地址，输入网址点击回车 --> 浏览器内部 DNS服务器先进行域名的映射，找到访问的地址，然后http客户端进程在80端口发起一个到服务器的tcp连接  
  --> http客户端通过它的套接字向服务器发送一个http请求报文，该报文包含了剩余路径的资源  
  --> http服务器通过套接字接收报文后进行请求的解析工作，并从存储器中检索出对象，将对象封装到HTTP响应报文中，通过套接字向客户进行发送。  
  --> HTTP服务器随即通知TCP断开连接，HTTP客户端接受完响应报文后TCP连接关闭。 --> 客户端从报文中提取响应文件，检查HTML文件，然后循环检查报文中其他内部对象。  
  --> 检查完成后，HTTP客户端会把对应的资源通过显示器呈现给用户。
